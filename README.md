# C# Training

A console-based C# practice repo for skills ramp-up. Each challenge lives in its own folder under `Challenges/`, with a small `Run()` entry point wired from `Program.cs`.

**Stack:** .NET 10 · C# · LINQ · Rider / VS Code

---

## Getting started

### Prerequisites

- [.NET SDK 10](https://dotnet.microsoft.com/download) (or the version in `csharp-training.csproj`)

### Run

```bash
cd csharp-training
dotnet run --project csharp-training/csharp-training.csproj
